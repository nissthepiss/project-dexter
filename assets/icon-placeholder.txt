# Icon Placeholder

You can create your own icon image and place it here.

## How to Create an Icon

### Option 1: Use an Online Tool
1. Go to https://icoconvert.com/
2. Upload a PNG image (256x256 or larger)
3. Download the `.ico` file
4. Place it in this folder as `icon.ico`

### Option 2: Use ImageMagick (if installed)
```bash
magick convert your-image.png -define icon:auto-resize=256,128,96,64,48,32,16 icon.ico
```

### Option 3: Quick Workaround
The app will build without an icon using the default Electron icon. 
Once you create one, replace this file and rebuild.

## Icon Requirements
- Format: `.ico` (Windows icon format)
- Size: 256x256 pixels minimum
- Should represent the app concept (charts, coins, tracking, etc.)
